# Telecom AI Demo

This 2-hour live demo shows how Artificial Intelligence and Machine Learning can be used in Telecom for:
- Predicting **call drops**
- Identifying **customer churn**

All work is done using **Jupyter Notebook**, and results are published to **GitHub**.
